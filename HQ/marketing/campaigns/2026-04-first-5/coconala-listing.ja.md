## 【よくある質問】（FAQ）

各質問を「質問を追加」ボタンから1つずつ登録。

### Q1
**質問:** なぜ¥1,000なんですか？相談カテゴリで出品している理由は？

**回答:**
```
Coconalaの「ホームページ作成」カテゴリは最低出品価格が¥20,000に設定されています。LP無料制作のお話をする最初のきっかけは¥1,000（ビール1杯分）にしたかったため、「IT相談」カテゴリでの出品としています。

ご購入いただくのはあくまで「LP制作前のご相談」です。実際の制作は相談後、月額¥5,000のホスティング契約をご検討いただける方に無料で進めさせていただきます。
```

### Q2
**質問:** 無料なのに月額¥5,000払う意味は何ですか？

**回答:**
```
サーバー・ドメイン管理・改善対応・セキュリティアップデートなど、サイト運営のランニングコストです。一般的なWeb制作会社では初期費用20〜50万円＋月額1〜3万円が相場ですが、その初期費用をZeroEn側で吸収する仕組みです。
```

### Q3
**質問:** 納期はどのくらいですか？

**回答:**
```
ヒアリング後、1〜3営業日で初稿をお渡しします。修正は1回まで無料です。
```

### Q4
**質問:** どんなビジネスが対象ですか？

**回答:**
```
個人事業主様・小規模事業者様・新規サービスのランディングページを想定しています。Eコマースや会員制サイトなどの複雑な要件は別メニュー扱いとなります。
```

### Q5
**質問:** 直接zeroen.devで申し込めますか？

**回答:**
```
将来的にはzeroen.devが中心になりますが、先着5名様の特別枠はCoconala経由のみで受付中です。
```

### Q6
**質問:** 購入前に相談できますか？

**回答:**
```
もちろんです。ビジネス概要を簡単にお聞かせいただければ、お作りできるかどうか事前にご回答します。お気軽にDMをお送りください。
```
