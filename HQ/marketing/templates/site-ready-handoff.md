# Site-Ready Handoff — Copy Templates

Bilingual (JP / EN) copy for two surfaces:
1. **Launch email** — operator sends when site is ready for client review
2. **Pre-payment block** — rendered above the plan/payment selector on zeroen.dev when the client's site is enabled

Replace `{{client_name}}` / `{{first_name}}` with the actual client name before sending.

## 1. Email — Japanese

**件名:** {{client_name}}様のサイト、公開準備が整いました

{{client_name}}様

お待たせしました。ご依頼いただいたウェブサイトの制作が完了し、公開できる状態になりました。一度ご確認いただき、デザインや文言など気になる点があれば遠慮なくお知らせください。

ここから先は「公開してからが本番」です。サイトを公開・運用していくために、月額のホスティングプランをご用意しています。月額費用には以下が含まれます:

- **ホスティング** — サーバー、SSL、独自ドメイン管理、稼働監視
- **月次アナリティクスレポート** — 訪問数・流入元・人気ページをPDFで毎月お届け
- **更新リクエスト** — プランに応じて、毎月の小さな修正・追加にも対応（文言修正、画像差し替え、セクション追加など）

サイトはただ作って終わりではなく、公開後に少しずつ育てていくものだと考えています。ホスティング費用はその「育てる仕組み」をまるごとお任せいただくためのものです。

プラン内容と金額は下記のページからご確認いただけます。気になる点があればお気軽にご返信ください。

→ [プランを選んでサイトを公開する]

引き続きよろしくお願いいたします。
ZeroEn
