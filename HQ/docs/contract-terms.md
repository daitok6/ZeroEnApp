# Contract Terms (Standard)

**Read this when:** Drafting a client engagement agreement, answering an ownership question, or handling cancellation/non-payment.

| Clause | Detail |
|--------|--------|
| **Pricing** | Fixed-price per tier. No hourly billing after kickoff. Scope and price locked in signed proposal. |
| **Code ownership** | ZeroEn retains all code rights. Client licenses the live site via active retainer. No buyout option in Phase 1 — will revisit for repeat/large clients. |
| **Scope freeze** | Deliverables locked at kickoff. Anything beyond = ¥15,000/hr, invoiced via Stripe. Ref: `HQ/crm/change-catalogue.md` |
| **Payment terms** | Per-tier milestone schedule (see `HQ/docs/revenue-model.md`). Retainer begins month after launch. |
| **Cancellation** | 30-day written notice required. Site archived on retainer lapse. |
| **Non-payment** | 14-day grace → retainer paused → 30 days → site archived. Reactivation requires clearing arrears. |
| **Refund policy** | 50% refundable before kickoff. 25% refundable within first week of build. No refunds after milestone 1 acceptance. |
| **Portfolio rights** | ZeroEn always retains right to showcase the work publicly. |
| **Domain** | Client-owned. ZeroEn manages DNS only during active retainer. |
| **Confidentiality** | Both parties keep business information confidential. Standard mutual NDA available on request. |
