## Growth Levers

1. **Referral program** — Existing clients refer new clients → discount on next change request
2. **Open source template** — Next.js + Supabase starter on GitHub → developers discover ZeroEn
3. **Product Hunt launches** — Launch each client app → dual exposure (client + ZeroEn)
4. **Build-in-public audience** — Compounding following on Twitter/X → organic inbound
5. **Portfolio effect** — More live apps = more proof = higher conversion

## Constraints

- Near-zero ad budget — organic and referral only
- Solo operator — no community management team
- No video — text/screenshots/code only
